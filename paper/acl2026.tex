\begin{abstract}
Large language models can iteratively paraphrase text while preserving semantic meaning, but it remains unclear how quickly and in what order authorial identity decays across paraphrase generations. We investigate this question using the Ship of Theseus Paraphrased Corpus, which contains texts paraphrased across four generations ($T_0 \rightarrow T_3$) by seven paraphraser variants spanning four model families. Combining lexical similarity metrics (BLEU, ROUGE-L), semantic similarity (BERTScore), POS tag distributions, and named entity recognition, we find that content-bearing features (named entities) decay 5.8$\times$ faster than stylistic features (POS distributions) as paraphrasing intensity increases. Semantic meaning is preserved (BERTScore 0.80--0.98) even as lexical surface features collapse (BLEU as low as 0.002), and named entities are predominantly \emph{dropped} rather than replaced by novel entities. Each paraphraser occupies a distinct position in the style-vs-content preservation space, with paraphraser configuration mattering more than model architecture. These findings have implications for plagiarism detection, AI-generated content identification, and computational forensics of authorial identity.
\end{abstract}
