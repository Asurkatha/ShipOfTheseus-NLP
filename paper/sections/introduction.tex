\section{Problem Description}

When you paraphrase a text multiple times using language models, something interesting happens: the writing gradually stops sounding like the original author. This project investigates exactly how this transformation occurs using the Ship of Theseus Corpus, which contains texts paraphrased through multiple iterations ($T_0 \rightarrow T_1 \rightarrow T_2 \rightarrow T_3$) by four different model families (ChatGPT, PaLM2, Pegasus, and Dipper). Like the philosophical paradox where replacing every plank of a ship raises the question of whether it is still the same ship, we examine when repeatedly paraphrased text loses its connection to the original author.

We explore three central questions: First, which linguistic features disappear fastest -- do stylistic markers like sentence structure and word choice vanish before semantic meaning, or vice versa? Second, is there a critical ``point of no return'' where authorship attribution suddenly fails, or does identity fade gradually across iterations? Third, can we identify which paraphrasing model was used by detecting model-specific patterns left behind in the text? Understanding these dynamics has real implications for detecting plagiarism, identifying AI-rewritten content, tracing misinformation sources, and understanding the fundamental boundary between human and machine authorship.
