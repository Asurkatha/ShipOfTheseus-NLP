# Processed Data

Preprocessed and cached corpus files in parquet format. Generated by running:

```bash
python -m src.data.prepare_corpus
```

## Directory Structure

```
processed/
├── corpus_full.parquet        # Full corpus (66,400 rows) for convenience
├── t0_human/                  # Original texts
│   ├── cmv.parquet
│   ├── eli5.parquet
│   ├── sci_gen.parquet
│   ├── tldr.parquet
│   ├── wp.parquet
│   ├── xsum.parquet
│   └── yelp.parquet
├── t1_paraphrased/            # First paraphrase iteration
│   └── ...
├── t2_paraphrased/            # Second iteration
│   └── ...
└── t3_paraphrased/            # Third iteration
    └── ...
```

## Columns

Each parquet file has the following columns:

| Column | Description |
|--------|-------------|
| `source` | Text producer (Human, OpenAI, LLAMA, Tsinghua, BigScience, PaLM, Eleuther-AI) |
| `key` | Document identifier |
| `text` | Cleaned text content |
| `version_name` | Raw version string from the corpus |
| `dataset` | Which of the 7 datasets this row belongs to |
| `split` | Always "test" for paraphrased data |
| `tier` | T0, T1, T2, or T3 |
| `paraphraser` | Paraphraser key (e.g., `chatgpt`, `dipper(low)`) |
| `paraphraser_name` | Display name (e.g., `ChatGPT`, `Dipper (Low)`) |
| `word_count` | Number of words |
| `char_count` | Number of characters |
| `sentence_count` | Approximate sentence count |
| `avg_word_length` | Mean word length in characters |
| `ttr` | Type-Token Ratio (lexical diversity) |
| `punct_density` | Punctuation characters / total characters |

## Preprocessing Steps

1. Removed PaLM2 markdown formatting artifacts
2. Normalized unicode whitespace
3. Filtered texts with fewer than 10 words
4. Preserved casing, punctuation, and sentence structure (needed for stylometry)
5. Deduplicated on (key, version_name)

## Usage

```python
import pandas as pd

# Load the full corpus
corpus = pd.read_parquet("data/processed/corpus_full.parquet")

# Or load a specific tier/dataset
t0_xsum = pd.read_parquet("data/processed/t0_human/xsum.parquet")
t3_yelp = pd.read_parquet("data/processed/t3_paraphrased/yelp.parquet")
```

## Note

Parquet files are git-ignored due to size. Regenerate them with `python -m src.data.prepare_corpus` after cloning the raw data.
